Bilan des Missions

Résumé des Missions

Mission 1 : Conception et Interrogation d'une Base de Données

Les principales tâches comprenaient la conception de tables, l’identification des clés primaires et étrangères, et la gestion des incohérences dans les données.

Mission 2 : Analyse de la Base de Données et Optimisation

Les objectifs étaient de résoudre les problèmes de données incohérentes, de performance des requêtes, et de faire des choix optimisés pour les données et les index.

Mission 3 : Requêtes Avancées

Cette mission impliquait des requêtes SQL avancées pour extraire des données pertinentes, comme les chiffres d'affaires par catégorie, les fournisseurs et clients stratégiques, et les analyses de marché.

Difficultés rencontrées

  • Incohérences dans les données, notamment pour les noms de villes et les formats des données.
  • Optimisation des requêtes pour des bases de données importantes, surtout pour des jointures complexes.
  • Utilisation de raccourcis et de logiciels non adaptés pour certaines tâches, comme Excel pour la gestion des relations.

Suggestions pour l'amélioration

  • Standardiser les formats de données et les noms pour réduire les incohérences.
  • Utiliser un logiciel de gestion de base de données adapté, comme MySQL ou Oracle, pour faciliter les opérations et améliorer la précision des résultats.
  • Optimiser les requêtes en créant des vues ou des fonctions pour des requêtes fréquemment utilisées.
  • Indexer les tables pour une meilleure performance dans les grandes bases de données.
